WasteWarden
Biomedical Waste Tracking & Compliance Platform

Overview
WasteWarden is a web-based platform designed to digitally track biomedical waste from generation to final disposal, ensuring real-time compliance, transparency, and regulatory oversight.
It connects hospitals, waste transporters, treatment facilities, and regulators through a unified system.

Problem Statement
 Biomedical waste is often managed using manual records and disconnected systems, which leads to:
 •	Improper disposal
 •	Delayed pickups
 •	Compliance violations
 •	Increased health and environmental risks
 There is a need for a centralized digital platform to monitor waste movement and ensure regulatory compliance in real time.

Objectives
 •	Track biomedical waste from generation to disposal
 •	Ensure real-time compliance with regulations
 •	Provide early alerts for violations and delays
 •	Generate audit-ready reports and dashboards

Technology Stack
 1.Frontend
  •	HTML
  •	CSS
  •	JavaScript
 2.Backend
  •	Python
  •	Django
 3.Database
  •	SQLite

 Key Features
 •	Waste logging by category and quantity
 •	Pickup request and status tracking
 •	Disposal and treatment record management
 •	Rule-based compliance checks
 •	Role-based dashboards (Hospital, Facility, Regulator, Admin)
 • GPS- enabled waste transport vehicles

Solution Workflow
 1.	Waste is logged at hospitals and laboratories
 2.	Pickup requests are generated and tracked
 3.	Waste is received at treatment facilities
 4.	Disposal method and timestamp are recorded
 5.	Compliance rules are validated
 6.	Regulators monitor dashboards and reports
